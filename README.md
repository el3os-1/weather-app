# 🌤️ Weather App

A simple and modern Weather Web App that shows real-time weather information for any city using WeatherAPI.

---

## 🚀 Live Demo
🔗 https://el3os-1.github.io/weather-app/

---

## ✨ Features

- 🔍 Search any city worldwide
- 🌡️ Real-time temperature
- 🌥️ Weather condition (Sunny, Rainy, Cloudy...)
- 💧 Humidity information
- 💨 Wind speed
- 📍 Get weather using your current location
- 🌙 Dark / Light mode toggle
- 🎨 Animated and modern UI

---

## 🛠️ Built With

- HTML5
- CSS3 (Glassmorphism + Animations)
- JavaScript (ES6)
- WeatherAPI
